# Scientific Calculator with Theme System

A fully functional and responsive Scientific Calculator built using HTML5, CSS3, and JavaScript.  

This project implements advanced mathematical operations along with a dynamic theme switching system for improved user experience.

## 🚀 Features

- Basic arithmetic operations (+, −, ×, ÷)
- Advanced scientific functions:
  - Trigonometric functions (sin, cos, tan)
  - Logarithmic functions (log, ln)
  - Square root and exponentiation
  - Percentage calculations
- Bracket support for complex expressions
- Real-time display updates
- Keyboard input support
- Error handling for invalid expressions
- Responsive layout (Desktop, Tablet, Mobile)
- Theme system (Light / Dark / Custom themes)
- real-time clock

## 🎨 Theme System

The calculator includes a dynamic theme switcher that allows users to toggle between multiple UI themes.  

The theme system is implemented using:
- CSS variables for color management
- JavaScript event listeners for theme switching
- LocalStorage to persist user theme preference

## 🛠️ Technologies Used

- HTML5 (Semantic Markup)
- CSS3 (Flexbox/Grid, Custom Properties)
- JavaScript (DOM Manipulation, Event Handling, Expression Parsing)
